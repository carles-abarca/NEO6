Cómo invocar transacciones

Las transacciones configuradas pueden invocarse a través de los endpoints REST del proxy:

  • POST /invoke: Invocación síncrona de una transacción.
  • POST /invoke-async: Invocación asíncrona.
  • GET /status/{id}: Consulta del estado/resultados de una invocación asíncrona.

Ejemplo de invocación vía curl:

curl -X POST http://localhost:8080/invoke \
  -H "Content-Type: application/json" \
  -d '{"transaccion": "CICS01", "parametros": {"cuenta": "12345"}}'

Consulta la documentación de cada protocolo para detalles de integración avanzada.
