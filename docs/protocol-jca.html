JCA / CICS TG

Conectividad Java EE mediante CICS Transaction Gateway, permitiendo la integración de aplicaciones Java empresariales con sistemas mainframe.

Ejemplo de definición en transactions.yaml

transactions:
  JCA01:
    protocol: "jca"
    server: "cics-tg-server"
    parameters:
      - name: "transaction"
        type: "string"
        required: true
      - name: "data"
        type: "object"
        required: true
    expected_response:
      status: "ok"
      fields:
        - name: "result"
          type: "object"
