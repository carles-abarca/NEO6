3 de junio de 2025 Guía de desarrollo v0.1.0 (neo6-proxy)

Guía de desarrollo

Cómo contribuir

  • Clona el repositorio y revisa la estructura de carpetas.
  • Implementa nuevos protocolos en neo6-protocols/ usando ProtocolHandler.
  • Agrega pruebas en tests/ y documentación en docs/.
  • Usa cargo test y cargo clippy para validar cambios.

Recomendaciones

  • Utiliza logging estructurado (tracing).
  • Documenta los endpoints y la configuración YAML/TOML.
  • Consulta README.md y docs/ para ejemplos y arquitectura.
