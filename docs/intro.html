Introducción a NEO6

NEO6 es un ecosistema de componentes para la interoperabilidad entre aplicaciones modernas y sistemas COBOL legacy (IBM z/OS, Micro Focus, etc.), permitiendo exponer transacciones CICS a través de APIs REST, MQ, TCP y otros protocolos.

  • Proxy de alto rendimiento escrito en Rust.
  • Soporte para múltiples protocolos y adaptadores.
  • Arquitectura modular y extensible.
  • Observabilidad, métricas y logging estructurado.

Esta documentación cubre la arquitectura, los módulos principales y la guía de desarrollo para contribuir o integrar NEO6 en tu entorno.
